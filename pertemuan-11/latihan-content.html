HTML

adalah singkatan dari HyperText Markup Language. HTML ini merupakan suatu sistem yang digunakan dalam internet untuk menentukan struktur konten pada halaman web.

CSS

CSS sendiri adalah singkatan dari Cascading Style Sheet. CSS merupakan tools pelengkap HTML yang digunakan untuk mengubah tampilan suatu situs.

Javascript

(disingkat JS) adalah bahasa pemrograman tingkat tinggi dan dinamis. Javascript populer di internet dan dapat bekerja disebagian besar web browser
